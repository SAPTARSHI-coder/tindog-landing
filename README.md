## 📖 Overview

TinDog is a meticulously crafted, Tinder-style app landing page. It showcases a highly polished, fully responsive marketing site built primarily with HTML5 and Bootstrap 5. The project layers custom gradient theming and visual refinements over a structured component-based layout to create a high-end, engaging user experience.

### ✨ Key Features

- **Dynamic Hero Section:** Immersive gradient background with app mockup, bold typography, and direct CTA download buttons (Apple / Google Play).
- **Benefit-Driven Feature Cards:** Clean, responsive 3-column feature highlights utilizing Bootstrap icons for clear visual communication.
- **Testimonial & Social Proof:** Centered pull-quote design paired with a circular cropped profile and a responsive media/press logo grid (TechCrunch, Mashable, etc.).
- **Tiered Pricing Structure:** Beautifully distinct Bootstrap pricing cards offering Chihuahua (Free), Labrador ($15/mo), and Mastiff ($29/mo) plans.
- **Robust Footer:** Multi-column navigation structure with a seamless gradient transition matching the hero section.

## 🛠 Tech Stack

| Technology | Implementation Role |
| :--- | :--- |
| **HTML5** | Semantic structure and page architecture. |
| **Bootstrap 5.3** | Core layout engine, grid system (`row-cols-1 row-cols-lg-3`), responsive utilities, and UI components (Cards, Buttons). |

---

## 🧠 Architectural Concepts

- **Section-Based Composition:** Logical compartmentalization using semantic `<section id="...">` anchors for scalable navigation.
- **Fluid Grid System:** Extensive use of Bootstrap's responsive grid (`flex-lg-row-reverse`, `row-cols-md-3`) for flawless viewing across mobile, tablet, and desktop viewports.
- **Utility-First Override:** Utilizing Bootstrap utilities to define spacing, typography, and layout, while applying selective custom CSS for brand-specific gradients and border-radius override.
